# utils

This library contains shared utility functions and constants for the gurokonekt monorepo.

## Utilities included:

- **country.utils.ts** - Country data, flags, phone codes, and timezone mappings
- **phone.utils.ts** - Phone number formatting and validation utilities  
- **form-validators.utils.ts** - Reusable form validation functions
- **stepper.utils.ts** - Multi-step form/stepper navigation utilities
- **display.utils.ts** - Display name helpers for UI formatting

## Running unit tests

Run `nx test utils` to execute the unit tests via [Jest](https://jestjs.io).
